# 模型 2 · 前缀和/前缀积 + 哈希

## 一句话识别

题目反复询问“某一段连续子数组的和、积或数量”时，考虑前缀模型。

前缀和表示：

```text
prefix = 从数组开头累加到当前位置的总和
```

如果当前前缀和是 `prefix`，希望某段子数组的和为 `k`，那么需要以前出现过：

```text
prefix - k
```

因为：

```text
当前前缀和 - 以前的前缀和 = 中间子数组的和
prefix - (prefix - k) = k
```

---

## 练习 2A：和为 K 的子数组（LeetCode 560）

实现：

```cpp
int subarraySum(vector<int>& nums, int k)
```

返回和等于 `k` 的连续非空子数组数量。

示例：

```text
输入：nums = [1,1,1], k = 2
输出：2
解释：[1,1] 可以从下标 0 开始，也可以从下标 1 开始
```

### 固定哈希关系

```text
key   = 以前出现过的前缀和
value = 该前缀和出现的次数
```

固定变量名：

```text
prefix = 当前前缀和
res    = 已找到的子数组数量
map    = 前缀和 → 出现次数
```

### 初始条件

遍历前先执行：

```cpp
map[0] = 1;
```

它表示“空前缀的和为 0，已经出现一次”。没有它，从数组开头就达到 `k` 的子数组会漏算。

### 每轮固定动作

```text
1. prefix 加上当前数字
2. 查询 prefix - k 以前出现了几次
3. 将这个次数加入 res
4. map[prefix]++，记录当前前缀和
```

必须先查询、再记录当前 `prefix`。

### 要求

1. 时间复杂度达到平均 `O(n)`。
2. 不使用两层循环。
3. 正确处理负数和 `k = 0`。
4. 在 `02A-和为K的子数组.cpp` 中完成。
