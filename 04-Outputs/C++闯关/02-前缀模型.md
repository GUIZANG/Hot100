# 模型 2 · 前缀和/前缀积 + 哈希

## 一句话识别

题目反复询问“某一段连续子数组的和、积或数量”时，考虑前缀模型。

前缀和表示：

```text
prefix = 从数组开头累加到当前位置的总和
```

如果当前前缀和是 `prefix`，希望某段子数组的和为 `k`，那么需要以前出现过：

```text
prefix - k
```

因为：

```text
当前前缀和 - 以前的前缀和 = 中间子数组的和
prefix - (prefix - k) = k
```

## 练习 2A：和为 K 的子数组（LeetCode 560）

实现：

```cpp
int subarraySum(vector<int>& nums, int k)
```

返回和等于 `k` 的连续非空子数组数量。

示例：

```text
输入：nums = [1,1,1], k = 2
输出：2
解释：[1,1] 可以从下标 0 开始，也可以从下标 1 开始
```

### 固定哈希关系

```text
key   = 以前出现过的前缀和
value = 该前缀和出现的次数
```

固定变量名：

```text
prefix = 当前前缀和
res    = 已找到的子数组数量
map    = 前缀和 → 出现次数
```

### 初始条件

遍历前先执行：

```cpp
map[0] = 1;
```

它表示“空前缀的和为 0，已经出现一次”。没有它，从数组开头就达到 `k` 的子数组会漏算。

### 每轮固定动作

```text
1. prefix 加上当前数字
2. 查询 prefix - k 以前出现了几次
3. 将这个次数加入 res
4. map[prefix]++，记录当前前缀和
```

必须先查询、再记录当前 `prefix`。

### 要求

1. 时间复杂度达到平均 `O(n)`。
2. 不使用两层循环。
3. 正确处理负数和 `k = 0`。
4. 在 `02A-和为K的子数组.cpp` 中完成。
5. 保存后提交“练习 2A 写完了”。

### 练习结果

独立编码并通过测试。

固定映射关系：

```text
key   = 以前出现过的前缀和
value = 该前缀和出现的次数
```

关键理解：

- `map[0] = 1` 表示数组开始前存在一个前缀和为 `0` 的虚拟起点。
- `map[prefix-k]` 可能大于 `1`，因为相同前缀和可能在多个位置出现。

## 练习 2B：除自身以外数组的乘积（LeetCode 238）

实现：

```cpp
vector<int> productExceptSelf(vector<int>& nums)
```

返回数组 `res`，其中：

```text
res[i] = nums 中除 nums[i] 以外所有元素的乘积
```

示例：

```text
输入：[1,2,3,4]
输出：[24,12,8,6]
```

### 限制

- 不允许使用除法。
- 时间复杂度必须是 `O(n)`。
- 除返回数组外，只使用 `O(1)` 额外空间。

### 两次遍历

第一次从左到右：

```text
prefix = 当前元素左边所有数字的乘积
先把 prefix 写入 res[i]
再执行 prefix *= nums[i]
```

第二次从右到左：

```text
suffix = 当前元素右边所有数字的乘积
先执行 res[i] *= suffix
再执行 suffix *= nums[i]
```

固定变量名：

```text
res    = 返回数组
prefix = 左侧乘积
suffix = 右侧乘积
```

### 要求

1. `res` 初始长度与 `nums` 相同。
2. `prefix` 和 `suffix` 初始值都为 `1`。
3. 正确处理数组中含有 `0` 的情况。
4. 在 `02B-除自身以外数组乘积.cpp` 中完成。
5. 保存后提交“练习 2B 写完了”。

### 练习结果

修正返回数组长度后通过。

关键理解：

```text
res[i] = i 左边所有元素的乘积 × i 右边所有元素的乘积
```

第一次遍历保存左侧乘积，第二次遍历乘上右侧乘积。必须先使用 `prefix/suffix`，再把当前元素乘进去，才能排除当前元素自身。

## 闭卷检测 2C：寻找数组的中心下标（LeetCode 724）

实现：

```cpp
int pivotIndex(vector<int>& nums)
```

中心下标 `i` 需要满足：

```text
i 左边所有元素之和 == i 右边所有元素之和
```

如果中心下标位于数组最左侧，则左侧和为 `0`；最右侧同理。存在多个答案时返回最左边的下标；不存在时返回 `-1`。

示例一：

```text
输入：[1,7,3,6,5,6]
输出：3
解释：1+7+3 = 5+6 = 11
```

示例二：

```text
输入：[1,2,3]
输出：-1
```

### 检测要求

1. 时间复杂度 `O(n)`。
2. 除几个整数变量外，不创建额外数组。
3. 自己设计如何利用已经遍历过的左侧和。
4. 在 `02C-前缀迁移检测.cpp` 中完成。

通过后再进行模型 2 的最终复测。

### 检测结果

修正空间复杂度、最左返回和 C++ 求和语法后通过。

关键顺序：

```text
suffix 先减去当前元素
比较 prefix 与 suffix
prefix 再加上当前元素
```
