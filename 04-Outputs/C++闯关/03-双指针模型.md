# 模型 3 · 双指针

## 三种常见形态

```text
同向快慢指针：原地筛选、覆盖、移动元素
左右对撞指针：从两端缩小搜索范围
不同速度指针：链表判环、找中点
```

这一模型先学习数组中的同向快慢指针，再学习左右对撞。链表快慢指针会在链表模块复习。

---

## 练习 3A：移动零（LeetCode 283）

实现：

```cpp
void moveZeroes(vector<int>& nums)
```

把数组中所有 `0` 移动到末尾，同时保持非零元素的相对顺序。必须原地修改数组，不能返回新数组。

示例：

```text
输入：[0,1,0,3,12]
修改后：[1,3,12,0,0]
```

### 快慢指针含义

固定变量名：

```text
fast = 正在检查的位置
slow = 下一个非零元素应该放置的位置
```

遍历时：

```text
如果 nums[fast] 非零：
    交换 nums[slow] 与 nums[fast]
    slow 向右移动
```

`fast` 每轮都向右移动，负责扫描整个数组。

### 简单过程

对于：

```text
[0,1,0,3,12]
```

开始 `slow=0`：

```text
fast 找到 1：与下标 0 交换，slow=1
fast 找到 3：与下标 1 交换，slow=2
fast 找到 12：与下标 2 交换，slow=3
```

最终得到：

```text
[1,3,12,0,0]
```

### 要求

1. 原地修改，不创建另一个数组。
2. 时间复杂度 `O(n)`。
3. 保持非零元素原顺序。
4. 在 `03A-移动零.cpp` 中完成。
5. 保存后提交“练习 3A 写完了”。
