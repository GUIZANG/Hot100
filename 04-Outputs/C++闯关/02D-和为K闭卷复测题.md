# 闭卷复测 2D · 和为 K 的子数组

## 题目

给定一个整数数组 `nums` 和一个整数 `k`，请返回数组中和等于 `k` 的连续非空子数组数量。

子数组必须是原数组中连续的一段。

## 函数签名

```cpp
int subarraySum(vector<int>& nums, int k)
```

## 示例一

```text
输入：nums = [1,1,1], k = 2
输出：2
```

## 示例二

```text
输入：nums = [1,2,3], k = 3
输出：2
```

满足条件的子数组是 `[1,2]` 和 `[3]`。

## 数据范围

```text
1 <= nums.length <= 20000
-1000 <= nums[i] <= 1000
-10^7 <= k <= 10^7
```

## 要求

1. 平均时间复杂度达到 `O(n)`。
2. 正确处理负数和 `0`。
3. 不查看旧答案 `02A-和为K的子数组.cpp`。
4. 在 `02D-和为K闭卷复测.cpp` 中完成代码。
